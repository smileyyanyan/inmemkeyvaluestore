This use case has two separate curl commands, one to save a new value and then to delete it. 

Save expected output: 
{"status":"OK"}


Delete expected output: 
{"status":"OK"}


=====================================================================
Save- - curl command to import to postman
=====================================================================

curl --location --request PUT 'localhost:9888/api/keyvaluestore/georgew' \
--header 'Content-Type: text/plain' \
--data '{"first_name": "George", "last_name": "Washington", "role": "President"}'


=====================================================================
Delete - curl command to import to postman
=====================================================================

curl --location --request DELETE 'localhost:9888/api/keyvaluestore/georgew'
